# AREC Constructions & Engineering Website

Bilingual (Arabic/English) single-page marketing website for AREC Constructions & Engineering.

## Setup

Open `index.html` in a browser. No build tools required.

## Structure

- `index.html` — Main single-page file
- `css/` — Stylesheets (variables, base, layout, sections, RTL, animations)
- `js/` — Vanilla JS modules (scroll, language, counters, nav, portfolio)
- `assets/` — Images and SVG icons
